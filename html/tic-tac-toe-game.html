Project Overview

The Tic-Tac-Toe Game with AI is a desktop application built using JavaFX that allows users to play against an AI opponent. The AI logic is designed to make intelligent moves, providing a challenging and engaging gameplay experience. The game features a clean and interactive UI, ensuring a smooth user experience.I have used minimax algorithm for that.

Key Features

  • Single-Player Mode: Play against an AI opponent.
  • Intuitive UI: Built with JavaFX .
  • Win/Loss/Draw Detection: The game automatically checks for a winner or a draw after each move..
  • Restart Option: Players can reset the game and start over.
